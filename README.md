# Plastic Invaders
In this short game coded for the SustainNYC hack-a-thon, Earth, the player, is being attacked by a barrage of plastic bottles. To fend off the enemeies, shoot biodegradable and eco-friendly bullets at them. Try your best and see what score you can get!

# Requirements
This game was developed on the LOVE2D software. Therefore, to run this game you must also have LOVE2D. <br/>The download is simple and it can be done so at:
https://love2d.org/
